# Project: Build an social media application like Twitter

## Auth
* Login
* Logout
* Signup
  
## App functions

### Admin user
* Create/Read/Delete/Update Users
* Create/Read/Delete Tweets
  
### Regular user
* post/delete tweets
* reply to posts (comments)
  
## Technology
* Frontend: React
* Backend: NextJS
* Database: firebasestore
  
## Deloyment: Netlify
